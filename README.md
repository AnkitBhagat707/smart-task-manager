# Smart Task Manager

## Overview
A full-stack task manager with JWT authentication and smart priority suggestion.

## Tech Stack
- Flask (Backend)
- React (Frontend)
- SQLite (Database)
- JWT (Auth)

## Features
- Signup & Login
- Secure APIs using JWT
- Create tasks
- Smart priority detection (High/Medium/Low)
- View tasks

## Run

Backend:
```
cd backend
python app.py
```

Frontend:
```
cd frontend
npm run dev
```
